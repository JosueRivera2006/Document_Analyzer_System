# Proyecto de Resumen de Documentos

Este proyecto fue desarrollado para la clase de **Lenguajes de Programación** y tiene como objetivo **resumir documentos extensos automáticamente**.  
Se apoya en la librería **Sumy**, que utiliza algoritmos para detectar las frases más importantes dentro de documentos grandes.

---

## Requisitos previos

Antes de ejecutar el proyecto, asegúrese de tener instaladas las siguientes librerías de Python:

```bash
pip install sumy nltk
pip install langdetect
pip install deep-translator
pip install wordcloud
pip install python-docx PyPDF2 python-pptx
pip install fastapi
pip install reportlab
```

## Funcionalidades implementadas

- [x] Lógica de resumen de documentos
- [x] Detección de idioma de documentos
- [x] Nube de palabras
- [x] Backend profesional
- [x] Método que reciba la ruta del archivo y aplique la lógica de resumen
- [x] Leer archivos PDFS, DOCS, TXT, POWERPOINT
- [x] Frontend intuitivo y fácil de usar
- [x] Opción para descargar el resumen en **Word**, **PPTX** **TXT** y **PDF**
- [x] Generación de Imágen de la Nube de Palabras
